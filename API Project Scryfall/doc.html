Process

I started with the gif finder js, and adapted the methods to work with Scryfall. I removed the ability to toggle amount of results, and instead added sorting by color and type, which use the AND operator. These used checkboxes. I then tested to make sure everything worked well with the searching, which was corect until I discovered problems with the double sided mtg cards. The images for such are stored within another nested object. This meant I had to do a check, then if the image didn't exist I went down to another object to access the images (Search 'tovolar' to see a few double sided cards), thank goodness for the google json viewer extension, made it easy to find what I needed. I also included the functionality of clicking on a card to make it appear larger in a spot, this allows the user to easily see a specific card. To do so I saved the normal version of each image and when the image was clicked, I applied that to the spot where the large image goes. In terms of CSS, I applied simple styles to make the cards appear in an orderly fashion, although with double sided cards, as they are nested together, can cause some visual problems.
